struct  Enumerator
 An enumerator for this type of array. It enumerates from (0,0) to (Length0-1,Length1-1) in rows of the first dimension then the second dimension. For example, an array with Length0=2 and Length1=3 is enumerated as follows: (0, 0) (1, 0) (0, 1) (1, 1) (0, 2) (1, 3) More...

Property Documentation

◆ IsCreated

bool IsCreated
get

Initially true when a non-default constructor is called but initially false when the default constructor is used. After Dispose is called, this becomes false. Note that calling Dispose on one copy of this object doesn't result in this becoming false for all copies if it was true before. This property should not be used to check whether the object is usable, only to check whether it was ever usable.
