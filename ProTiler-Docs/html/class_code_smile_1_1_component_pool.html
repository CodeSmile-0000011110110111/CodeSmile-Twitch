ComponentPool< T > Class Template Reference

Pools a component of a prefab. This also works with GameObject instances except that you will have to pool Transform component instead and access GO via gameObject property. CAUTION: remember to call Dispose() or use a using() statement to ensure pooled instances get destroyed. Failure to do so will lead to stray objects in the pool's parent or throw an exception if the finalizer runs. More...
